# SSF2022 Ｘ EDCAN
2022 소프트웨어나눔축제 EDCAN 캠프 예제를 위해서 제작한 일기장앱 입니다.

# 제작
- 박희찬

# 사용 기술
- Kotlin
- Android Studio
- MVVM (LiveData, DataBinding)
- Coroutine
- Firebase

# EDCAN 이란
